# Blockchain Simple Implementation

## 区块链的哲学意义

区块链是一个优美的分布式系统, 使得<u>互相不信任的个体</u>间, 对某种状态机能够达到稳定的共识.

从而<u>不依靠任何中心化权威机构</u>, 制造了<u>对这个共同维护的状态机的高强度信任</u>.

=> **"信任制造机"**
